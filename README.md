# ⚡ Towards Dev

A smart productivity tracker designed for students preparing for **GATE**, **AIML**, coding interviews, and personal growth.

Track daily goals, maintain learning streaks, visualize progress, and stay consistent on your journey toward success.

---

## 🚀 Features

### ✅ Daily Task Tracking

* Mark tasks as completed with a single click.
* Progress is automatically saved in the browser.

### 📈 Progress Ring

* Visual percentage-based progress tracker.
* Updates instantly as tasks are completed.

### 🔥 Streak System

* Builds streaks when all daily tasks are completed.
* Encourages consistency and discipline.

### 📅 Weekly Heatmap

* View activity and productivity for the last 7 days.
* Quickly identify productive and unproductive days.

### 🎯 Goal Categories

Tasks can be organized into:

* 🟡 LeetCode
* 🔵 GATE
* 🟣 AIML
* 🟢 Weekend Tasks
* 🌸 Bonus Habits

### ⚙️ Admin Panel

Protected admin area with:

* Add custom tasks
* Delete tasks
* Manage schedules
* Change admin password
* Reset all stored data

### 💾 Offline Support

Works even without an internet connection using:

* Service Workers
* Browser Cache Storage

### 📱 Installable PWA

Install the app directly on:

* Android
* Windows
* macOS
* Linux

and use it like a native application.
